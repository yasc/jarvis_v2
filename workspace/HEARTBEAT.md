# HEARTBEAT.md

## Morning Check-in (if run between 6:00-9:00)
1. Check today's calendar events (use gog calendar)
2. Check weather for London
3. Check for any unread important emails in the last 12 hours
4. Check pending reminders due today
5. Summarise briefly. Lead with time-sensitive items.
6. If nothing notable, reply HEARTBEAT_OK

## Midday Pulse (if run between 11:00-13:00)
1. Check for any new emails since morning
2. Check for upcoming events in the next 4 hours (use gog calendar)
3. Check if any reminders are due in the next 4 hours
4. Only report if something needs attention. Otherwise HEARTBEAT_OK

## Evening Wind-down (if run between 17:00-19:00)
1. Check tomorrow's calendar (use gog calendar)
2. Check pending reminders for tomorrow
3. Clean up old fired/cancelled reminder entries if there are more than 50
4. If nothing, HEARTBEAT_OK
